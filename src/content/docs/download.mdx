## macOS

Wuhu for macOS is a notarized app distributed as a `.zip` archive. Download,
unzip, and drag to your Applications folder. The app checks for updates
automatically using [Sparkle](https://sparkle-project.org/) — you'll get a
native update prompt whenever a new version is available.

### Installation

1. Download the `.zip` file above
2. Unzip it (double-click or `unzip`)
3. Drag **Wuhu.app** into your `/Applications` folder
4. Open Wuhu from Applications — on first launch, macOS may ask you to confirm
   since the app was downloaded from the internet

### Updating

Wuhu includes automatic updates. When a new version is available, you'll see a
native update dialog. You can also check manually from the **Wuhu → Check for
Updates…** menu.
